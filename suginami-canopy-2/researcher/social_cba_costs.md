# 26haシナリオの社会的費用便益分析: 費用計算ロジック

## 1. このメモの前提

このメモでは、杉並区で追加的に確保する成熟樹冠・グリーンインフラ機能を **26ha** と置いた場合に、社会全体でどのような費用が発生するかを整理する。

社会的費用便益分析では、杉並区役所の支出だけではなく、区民、土地所有者、事業者、来街者を含む社会全体で発生する実費、維持管理負担、機会費用、リスク対応費を考える。

一方で、補助金そのものは注意して扱う。補助金は、行政から住民・事業者へのお金の移転であり、社会全体で見ればそれ自体は純費用ではない。社会的費用として見るべきなのは、実際の植栽、土壌改良、維持管理、所有者の自己負担、土地や空間の機会費用である。

金額は、原則として **30年・割引率4%の現在価値** で示す。現在価値とは、将来発生する費用を現在の価値に割り引いて合計した値である。本試算では、毎年同じ費用が30年間発生する場合の換算係数を **17.292** としている。

```text
30年・4%の現在価値係数
= (1 - (1 + 0.04)^-30) / 0.04
= 17.292
```

また、本シナリオでは面積を次のように分けて考える。

| 記号 | 意味 | 26ha中心ケースでの値 |
|---|---|---:|
| 成熟樹冠面積 | 木の枝葉が成熟後に上空から覆う面積。暑熱、健康、景観便益の対象 | 26ha |
| 地表GI面積 | 植樹帯、雨庭、浸透施設、透水性舗装などを実際に整備する地表面積。費用の対象 | 13ha |
| 集水・処理面積 | 雨庭や浸透施設が雨水を受け止める面積。雨水便益の対象 | 26ha |

中心ケースでは、成熟した樹冠は植栽地表面より広がると考える。そのため、成熟樹冠26haを得るために、実際に整備する地表GI面積は13haと置く。

```text
地表GI面積
= 成熟樹冠面積26ha ÷ 樹冠/地表GI比率2.0
= 13ha
```

## 2. 費用ケースの全体像

費用は、用地取得を伴う単独緑地整備か、既存の公園・学校・道路・公共施設の改修に組み込むかで大きく変わる。本シナリオの中心は、追加用地取得ではなく、区有地・公共空間・公共性の高い民有地を活用するケースである。

| ケース | 地表GI面積 | 費用PV | 位置づけ |
|---|---:|---:|---|
| 重点実装・中心 | 13ha | 約2.79億円 | 本体CBAの中心候補 |
| 標準費用感度 | 13ha | 約4.76億円 | 費用をやや厚めに見た感度 |
| 単独用地取得型 | 13ha | 数百億円規模 | 中心から除外。CBAはほぼ成立しない |

中心ケースの費用は、地表GI面積13haに対し、初期整備費、維持管理費、行政・リスク費を計上して作る。

## 4. 中心ケースの費用計算

### 4.1 何を中心ケースとするか

中心ケースは、道路、学校、公園、公共施設、公共性の高い民有地の改修・活用にグリーンインフラを組み込むケースである。ここでは、通常の施設整備や道路整備の全費用ではなく、樹冠、雨庭、浸透施設、透水性舗装などを追加するための増分費用を見る。

中心ケースでは、地表GI面積13haに対し、初期整備費を10百万円/ha、維持管理費を0.5百万円/ha・年、行政・リスク費を15%と置く。

| 費用要素 | 計算 | 26ha費用PV |
|---|---|---:|
| 初期整備費 | 13ha × 10百万円/ha | 約1.30億円 |
| 維持管理費 | 13ha × 0.5百万円/ha・年 × 17.292 | 約1.12億円 |
| 行政・リスク費 | 上記小計 × 15% | 約0.36億円 |
| 合計 |  | 約2.79億円 |

計算式で示すと、次の通りである。

```text
初期整備費
= 13ha × 10百万円/ha
= 130百万円

維持管理費PV
= 13ha × 0.5百万円/ha・年 × 17.292
= 112.4百万円

行政・リスク費
= (130百万円 + 112.4百万円) × 15%
= 36.4百万円

合計費用PV
= 130百万円 + 112.4百万円 + 36.4百万円
= 278.8百万円
= 約2.79億円
```

### 4.2 中心ケースで含めているもの

中心ケースの初期整備費には、高木植栽、土壌改良、植樹基盤、雨庭、浸透施設、透水性舗装、支柱、初期灌水設備、外構調整などを含む。

維持管理費には、剪定、灌水、施肥、病害虫対応、落葉清掃、雨庭・浸透施設の清掃、枯死更新の一部を含む。

行政・リスク費には、設計・調査、申請・審査、現地確認、台帳更新、モニタリング、苦情対応、事故・枯死・根上がりなどの予備費を概括的に含む。

### 4.3 中心ケースで含めていないもの

中心ケースでは、次の費用は原則として含めていない。

- 新たに土地を市場価格で買う費用。
- 市場賃料に近い借地料。
- 学校、道路、公園、公共施設の通常改修費そのもの。
- 大きな交通容量低下や駐車場喪失の機会費用。
- 住民や所有者の未補助労力を詳細に金額化したもの。
- 大規模工事による長期通行規制、営業損失、騒音被害。

これらは、候補地ごとに発生する可能性がある。中心ケースでは、そうした費用が大きくならない場所を選ぶ前提である。

## 5. 標準費用感度

中心ケースは、重点実装により費用を抑えられる場合のケースである。より慎重に見るため、初期費用と維持管理費を厚めに置く標準費用感度も設定する。

標準費用感度では、地表GI面積13haに対し、初期整備費を18百万円/ha、維持管理費を0.8百万円/ha・年、行政・リスク費を15%と置く。

| 費用要素 | 計算 | 26ha費用PV |
|---|---|---:|
| 初期整備費 | 13ha × 18百万円/ha | 約2.34億円 |
| 維持管理費 | 13ha × 0.8百万円/ha・年 × 17.292 | 約1.80億円 |
| 行政・リスク費 | 上記小計 × 15% | 約0.62億円 |
| 合計 |  | 約4.76億円 |

```text
初期整備費
= 13ha × 18百万円/ha
= 234百万円

維持管理費PV
= 13ha × 0.8百万円/ha・年 × 17.292
= 179.8百万円

行政・リスク費
= (234百万円 + 179.8百万円) × 15%
= 62.1百万円

合計費用PV
= 234百万円 + 179.8百万円 + 62.1百万円
= 475.9百万円
= 約4.76億円
```

この標準費用感度は、費用が中心ケースより高く出た場合でも政策判断が大きく崩れないかを見るためのものである。

## 6. 用地・空間確保費の扱い

### 6.1 なぜ中心ケースでは用地費を0円と置くのか

中心ケースでは、すでに公共目的で使われている区有地・公共空間、または公共性の高い民有地を活用する。つまり、グリーンインフラ政策のために新たに土地を市場価格で買う前提ではない。

このため、中心ケースでは用地取得費を0円と置く。ただし、これは「土地に価値がない」という意味ではない。既存の公園、学校、道路、公共施設、民有地には、それぞれ別の用途がある。社会的費用便益分析では、土地や空間の代替利用を妨げる場合、その機会費用を考える必要がある。

中心ケースでは、校庭、避難所機能、歩道幅員、消防活動空地、既存公園機能を大きく損なわない場所を選ぶため、機会費用を小さいものとして扱っている。

### 6.2 市場取得の場合の感度

既存分析では、杉並区住宅地の地価水準を参考に、土地費proxyを **約703,177円/m2** と置いている。これは1haあたりでは約70.3億円である。

```text
1haあたり土地費
= 703,177円/m2 × 10,000m2
= 約7,032百万円
= 約70.3億円
```

地表GI面積13haを市場価格で取得すると、土地費だけで次の規模になる。

```text
13haの土地費
= 約70.3億円/ha × 13ha
= 約914億円
```

この場合、便益PVが約11.55億円から13.50億円であるのに対し、用地費だけで数百億円規模になるため、CBAはほぼ成立しない。

### 6.3 借地の場合の感度

土地を買わずに借りる場合でも、市場賃料に近い借地料を払うと費用は大きくなる。既存分析では、地価の0.5%/年、1.0%/年、2.0%/年を借地料感度として置いている。

| 借地料仮定 | 1haあたり年額 | 1haあたり30年PV | 評価 |
|---:|---:|---:|---|
| 0%/年 | 0百万円 | 0百万円 | 市民緑地契約の無償貸付に近い |
| 地価の0.5%/年 | 約35.2百万円 | 約608百万円 | 便益PVを大きく上回りやすい |
| 地価の1.0%/年 | 約70.3百万円 | 約1,216百万円 | 市場性を入れるとCBAはほぼ成立しない |
| 地価の2.0%/年 | 約140.6百万円 | 約2,431百万円 | GI施策としては過大 |

したがって、民有地活用は、無償・低廉貸付、税優遇、管理代行、所有者の地域貢献意向が成立する案件に限定するのが望ましい。

## 7. 民有地補助の扱い

民有地への補助は、財政分析では杉並区の支出である。しかし、社会的費用便益分析では、補助金そのものは行政から所有者への移転であり、社会全体の純費用ではない。

社会的費用として見るべきものは、次の通りである。

| 見るべき費用 | 例 |
|---|---|
| 実施工費 | 苗木、植栽、土壌改良、支柱、雨庭、透水性舗装 |
| 維持管理費 | 剪定、灌水、落葉清掃、病害虫対応、枯死更新 |
| 所有者・住民負担 | 自己負担額、水やり、清掃、近隣対応、管理委託 |
| 行政運営費 | 申請審査、現地確認、成果確認、台帳更新、苦情対応 |
| 機会費用 | 駐車場、庭、通路、建替え余地など他用途の喪失 |

したがって、社会的CBAでは、民有地補助を費用分類8にそのまま足すのではなく、実際に発生する施工費、維持管理費、所有者負担、行政運営費に分けて計上する。

一方、財政分析では、補助金や税制優遇は区の支出・減収として別途扱う。

## 8. 本体に十分入っていない費用

次の費用は重要だが、現時点では中心ケースの中で概括的にしか扱っていない。候補地台帳ができた段階で、独立項目として再計算する必要がある。

| 費用 | 現時点の扱い | 今後必要なデータ |
|---|---|---|
| 枯死・更新費 | 行政・リスク費15%の内数 | 樹種別生存率、更新周期、再植栽単価 |
| 根上がり・舗装補修 | 行政・リスク費15%の内数 | 道路・歩道別の補修件数、根系対策費 |
| 落葉清掃・苦情対応 | 維持管理費の内数 | 清掃頻度、苦情件数、委託単価 |
| 所有者・住民の未補助負担 | 未分離 | 作業時間、自己負担額、剪定費、清掃費 |
| 交通・駐車の機会費用 | 中心では原則0円 | 失われる駐車台数、交通容量、移動時間 |

| 事故・賠償・保険 | 行政・リスク費15%の内数 | 点検頻度、保険料、事故件数、賠償額 |

これらを全て独立に足すと、行政・リスク費との二重計上になる可能性がある。現時点では概括的に15%を置き、詳細データが得られた段階で内訳を置き換えるのが安全である。

## 9. 費用便益比への接続

便益側の26haシナリオでは、政策判断用本体便益を約11.55億円PV、拡張便益を約13.50億円PVとしている。これを費用ケースに接続すると、次の通りである。

| 便益セット | 便益PV | 中心費用2.79億円でのB/C | 標準費用4.76億円でのB/C |
|---|---:|---:|---:|
| 政策判断用本体便益 | 約11.55億円 | 約4.14 | 約2.43 |
| 拡張便益 | 約13.50億円 | 約4.84 | 約2.84 |
| 参考上限 | 約37.25億円 | 約13.36 | 約7.83 |

この結果は、土地を新たに市場価格で取得せず、既存改修や公共空間活用に組み込める場合の試算である。追加用地取得や高額借地が必要になると、費用は桁違いに増え、B/Cは大きく低下する。

## 10. まとめ

26haシナリオの社会的費用は、中心ケースでは **約2.79億円PV**、標準費用感度では **約4.76億円PV** と置く。

この費用は、成熟樹冠26haそのものに対してではなく、実際に整備する地表GI面積13haに対して計算している。中心ケースの考え方は、既存の公園、学校、道路、公共施設、公共性の高い民有地を活用し、通常改修費や用地取得費を二重計上せず、グリーンインフラに固有の増分費用を計上するものである。

最も重要な注意点は、用地費である。市場価格で土地を買う、または市場賃料に近い借地料を払う場合、費用は数百億円規模になり、社会的費用便益分析は成立しにくい。したがって、政策設計上は、区有地・公共空間の活用、既存改修への組み込み、公共性の高い民有地の無償・低廉活用を中心に据える必要がある。

## 11. 外部引用・計算ロジック

この費用試算は、26haの成熟樹冠を得るために必要な実整備面積を13haと置き、その地表GI面積に対して初期整備費、維持管理費、行政・リスク費を積み上げたものである。

### 11.1 共通の現在価値計算

維持管理費など毎年発生する費用は、30年・割引率4%で現在価値化している。

```text
30年・4%の現在価値係数
= (1 - (1 + 0.04)^-30) / 0.04
= 17.292
```

### 11.2 地表GI面積13haの考え方

費用は成熟樹冠26haではなく、実際に整備する植樹帯、雨庭、浸透施設、透水性舗装などの地表GI面積13haに対して計算している。

```text
地表GI面積
= 成熟樹冠26ha ÷ 樹冠/地表GI比率2.0
= 13ha
```

この「樹冠/地表GI比率2.0」は、成熟した樹冠が植栽基盤より広がることを反映した政策仮定である。候補地設計段階では、樹種、植栽間隔、樹冠成熟年数、生存率を入れて置き換える必要がある。

外部出典:
- 国土交通省「[グリーンインフラについて](https://www.mlit.go.jp/sogoseisaku/environment/sosei_environment_fr_000192.html)」

### 11.3 初期整備費・維持管理費

中心ケースでは、地表GI面積13haに対し、初期整備費10百万円/ha、維持管理費0.5百万円/ha・年、行政・リスク費15%を置く。

```text
中心ケース費用PV
= 初期整備費13ha × 10百万円/ha
 + 維持管理費13ha × 0.5百万円/ha・年 × 17.292
 + 行政・リスク費15%
= 約2.79億円
```

標準費用感度では、初期整備費18百万円/ha、維持管理費0.8百万円/ha・年を置く。

```text
標準費用PV
= 初期整備費13ha × 18百万円/ha
 + 維持管理費13ha × 0.8百万円/ha・年 × 17.292
 + 行政・リスク費15%
= 約4.76億円
```

これらの単価は、杉並区で入札単価を確認した値ではなく、政策シナリオ用の増分費用仮定である。グリーンインフラは、雨水貯留・浸透、緑陰、ヒートアイランド対策など複数機能を持つため、通常改修に上乗せされる増分費用として扱う。

### 11.4 用地費・借地料の感度

本体ケースでは、追加用地取得を行わず、既存の区有地、公共空間、公共性の高い民有地を活用するため、用地取得費を0円と置く。市場価格で土地を取得する場合は、費用が桁違いに大きくなる。

土地費感度では、杉並区住宅地の地価水準を1m2あたり約703,177円として使っている。

```text
1haあたり土地費
= 703,177円/m2 × 10,000m2
= 約70.3億円

13haの土地費
= 約70.3億円/ha × 13ha
= 約914億円
```

このため、用地取得型は26haシナリオの中心ではなく、成立しにくい感度として扱う。

### 11.5 補助金の扱い

社会的費用便益分析では、補助金そのものは行政から住民・事業者への移転であり、社会全体の純費用ではない。社会的費用として見るべきなのは、実際の施工費、維持管理費、所有者負担、行政運営費、機会費用である。

一方、財政分析では、民有地緑化補助は杉並区の歳出である。したがって、社会的CBAと財政分析で扱いを分ける。
