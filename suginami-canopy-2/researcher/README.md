# 杉並区樹冠率向上政策の分析メモ

このメモは、杉並区で樹冠・緑被を追加的に確保する政策について、便益・費用の分類、26haシナリオ、社会的費用便益分析、杉並区財政への影響を整理するものである。

詳細な計算ロジックは、以下の補助資料に分けて整理している。

## 1. 分析視点

樹冠率向上政策は、誰にとっての費用・便益を見るかによって結論が変わる。そのため、本メモでは次の3つを分けて扱う。

| 分析視点           | 見るもの                                     | 目的                                 |
| ------------------ | -------------------------------------------- | ------------------------------------ |
| 社会的費用便益分析 | 区民・事業者・行政を含む社会全体の費用と便益 | 政策が社会全体として望ましいかを見る |
| 財政分析           | 杉並区役所の支出・収入・将来コスト削減       | 区の予算上、実行可能かを見る         |
| 住民負担分析       | 住民個人・世帯が受ける負担と便益             | 負担の公平性、受容性、分配影響を見る |

本メモでは、社会的費用便益分析と杉並区財政分析を中心に整理する。住民負担分析は、民有地補助、維持管理、家賃・地価上昇、分配面の論点として補足的に扱う。

## 2. 26haシナリオ

杉並区の緑被率25%の目標に対して、公園・緑地整備等を見込んだ後も追加的に必要な26haを十分に補うように、区有地・公共空間の活用を中心に据え、民有地活用の取り組みも3段階に分けたシナリオを考えた。

なお、杉並区の計画である25%達成に必要な追加緑被地約102haに対し、今後、公園・緑地等として整備される見込みの公共系緑地約76haを差し引いた不足分が、おおむね26haである。

**「住環境価値向上」を金銭便益として入れる場合、暑熱緩和・景観・歩きやすさなどの効果が地価にすでに反映されている可能性がある**ため、杉並区の政策評価では、基本ケースとしては以下のように分けるのが堅実です。

#### 3.1.2 便益の参入方針

| 扱い                 | 便益                                                       |
| -------------------- | ---------------------------------------------------------- |
| 主便益として金銭換算 | 暑熱健康被害削減、冷房費削減、雨水流出抑制                 |

つまり、費用便益分析の中核は **「熱」「電力」「雨水」** に置き、**「地価・住環境価値」は二重計上に注意して別枠で示す**のが最も説明しやすい構成です。

#### 3.2.2 補助金の扱い

行政会計上は、民有地緑化補助は「費用」です。しかし社会的費用便益分析では、補助金そのものは行政から住民・事業者への**移転**であり、社会全体の純費用ではありません。社会的費用として見るべきなのは、実際に発生する植栽費・維持管理費・機会費用です。

## 4. 社会的費用便益分析
社会的費用便益分析上は、26haシナリオは 便益が費用を上回る可能性が高い と評価できる。
特に、区有地・公共空間・公共性の高い民有地を使い、学校・公園・道路・公共施設の改修と一体で実施できる場合、**B/Cは本体便益でも約2.4〜4.1程度**。

ただし、この結論は **追加用地取得をしない、または市場賃料を払わない** ことが重要な前提です。市場価格で土地を買う・借りる場合、費用は数百億円規模になり、CBAは成立しにくくなります。

## 5. 杉並区財政への影響

26haシナリオは、杉並区財政だけを見ると、短期的には純支出になる可能性が高い。

確度が高いのは、植樹・植栽整備費、維持管理費、行政運営費、民有地緑化補助などの支出である。一方、区有施設の冷房費削減などの直接的な歳出削減はあるが、金額規模は小さい。したがって、財政分析の本体では、区財政に対しては赤字、すなわち純支出として扱うのが妥当である。

ただし、中長期的に区財政へ好影響が想定できないわけではない。樹冠、日陰、歩行快適性、公園・学校・道路環境の改善は、現役世代・子育て世代に選ばれ続ける住環境の一部になりうる。その結果、転出抑制・定住促進を通じて特別区民税を維持する可能性がある。

この効果は不確実性が高いため、参考分析として扱う。

### 5.1 財政影響の基本整理

| 区分 | 内容 | 財政分析での扱い |
|---|---|---|
| 短期の支出 | 植樹、植栽基盤、維持管理、補助、行政運営 | 本体に入れる |
| 短期の直接リターン | 区有施設の冷房費削減 | 本体に入れるが小さい |
| 雨水・暑熱・健康便益 | 社会的には重要だが、区歳入や区支出削減に直結しにくい | 区財政には原則入れない |
| 固定資産税・地価上昇 | 23区では固定資産税は都税 | 杉並区の直接歳入にしない |
| 生産年齢人口維持 | 住環境改善による転出抑制・定住促進 | 参考分析として扱う |

標準的な実装ケースでは、中心費用と標準費用の中間を取り、外部財源・民間負担を30%程度見込むと、区負担は約2.66億円PVとなる。区有施設の冷房費削減を約0.30億円PVと置くと、短期・直接財政だけでは約2.36億円PVの純支出となる。

```text
直接財政インパクト
= 区有施設の冷房費削減0.30億円 - 区負担2.66億円
= 約-2.36億円
```

したがって、26haシナリオは「区財政だけで短期に黒字化する政策」ではなく、「区の実質負担を管理可能な範囲に抑えながら、社会便益を得る公共投資」と説明するのが適切である。

### 5.3 研究上の支持と限界

生産年齢人口維持効果については、「樹冠率向上政策により自治体全体の人口減少が何%抑制された」と直接示す強い研究は限られる。そのため、上記の0-10%は効果推計ではなく、財政感度分析である。

一方で、住まいの周辺の緑は、生活満足度、転居意向、住宅価格、地域属性変化と関係することが複数の研究で示されている。したがって、樹冠・緑被政策が住環境魅力を通じて定住促進に寄与しうるという方向性は支持される。

ただし、これらの研究は「杉並区で生産年齢人口が何%維持されるか」を直接示すものではない。そのため、財政分析では0%を本体、1-2%を参考分析、5%以上を上振れ感度として扱う。研究ごとの主張と限界は、`fiscal_impact_analysis.md` に整理する。

以上から、財政分析では次の整理が妥当である。

- 短期・直接財政では、26haシナリオは純支出として扱う。
- 生産年齢人口維持は、本体ではなく参考分析にとどめる。
- 参考分析の中心は1-2%程度とし、5%は住環境政策パッケージの上振れ、10%は参考上限とする。
- 地価上昇や固定資産税増は、杉並区の直接歳入として扱わない。
- 人口維持を期待する場合でも、家賃上昇や低所得世帯の転出など、分配面の副作用を監視する必要がある。

つまり、財政分析上の結論は次の通りである。

> 26haシナリオは、短期の杉並区財政だけで見ると純支出である。しかし、樹冠・日陰・歩行環境・学校や公園の質を高めることが、現役世代・子育て世代に選ばれ続ける住環境の一部となる場合、中長期的には特別区民税を維持する可能性がある。この効果は不確実であるため本体には入れないが、参考分析として1-2%程度の生産年齢人口維持効果を置くことは可能である。
